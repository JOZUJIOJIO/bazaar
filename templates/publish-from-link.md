# Remix from Link Checklist

1. [ ] Run `bazaar steal <url>` to fetch content
2. [ ] Read the fetched content and identify the core workflow
3. [ ] Create a new skill folder with `SKILL.md` using `templates/skill-template.md`
4. [ ] Make the skill self-contained (no external dependencies on the original)
5. [ ] Add provenance marker: `<!-- remixed from <url> -->`
6. [ ] Test the skill locally
7. [ ] Publish: `bazaar publish ./new-skill --tags tag1,tag2`
